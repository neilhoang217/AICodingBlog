AZURE APP SERVICE SETUP (Portal Steps)
=======================================

STEP 1 — Create a Resource Group
----------------------------------
1. Go to portal.azure.com
2. Search for "Resource groups" in the top search bar
3. Click Create
4. Fill in:
   - Subscription: your subscription
   - Resource group name: AICodingBlog-rg
   - Region: pick the closest to you (e.g. East US)
5. Click Review + create -> Create


STEP 2 — Create the App Service
---------------------------------
1. Search for "App Services" in the top search bar
2. Click Create -> Web App
3. Fill in:
   - Resource group: AICodingBlog-rg
   - Name: something unique like "aicoding-blog"
     (this becomes aicoding-blog.azurewebsites.net)
   - Publish: Code
   - Runtime stack: .NET 10
   - OS: Windows
   - Region: same as your resource group
4. Under Pricing plan -> click Explore pricing plans -> select F1 (Free)
5. Click Review + create -> Create

NOTE: Deployment takes about 2 minutes.
After creation, note down the App Service name you chose.
You will need it to wire up GitHub Actions deployment.
